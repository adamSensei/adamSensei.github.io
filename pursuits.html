PURSUITS

PURSUITS is a CRM for career transitions, targeted at both higher education and professionals looking for a change. The application will allow users to easily manage and streamline their day to day tasks and schedule.

Role

UI, Branding



Process

Branding, UI Kit, Hi Fidelity Mockups



Goal

To create new brand and user interface for PURSUITS.



UI

PURSUITS approached me looking to simplify their user interfance in order to make the experience as seamless as possible.

Branding

The main purpose of PURSUITS is to keep job seekers organized and on task, so a clean and inviting color scheme was chosen. The majority of the UI sports nuetral colors, while the main focus of the application, the cards, are encased with a soothing tan. The tan color connects the user with the well known manila folder, a symbol of organization.

Mockups

The mockups show both the main dashboard and the job posting views. The UI animation demonstrates a user easily adding a job from the suggested section.
