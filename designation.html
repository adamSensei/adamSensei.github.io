DESIGNATION Job Board

DESIGNATION asked me to create an internal job board to better connect students with positions in the digital design community. For the project, I was paired with a back end developer and was tasked with creating a responsive front end experience.

Role

UI, Ixd, FED



Process

Wireframes, User Testing, HTML, CSS, JavaScript, JQuery



Goal

To design and develop a job board for the DESIGNATION Labs community.

CHECK OUT THE CODED SITE

IxD

Taking inspiration from other job boards across the web, I created a refined version that met the needs of the DESIGNATION community. While building the front end of the application, students and mentors constantly evaluated the layout, colors, and hierarchy for usability.

Wireframes

Before coding, I created wireframes for the desktop and mobile versions of the job board.

FED

I used the Sublime text editor and coded most of the website from scratch. Using media queries I made the web application responsive.

My partner used Meteor for the backend, so I needed to template my code in order to make it compatible.

Mix It Up

I used the Mix It Up plugin from Kunka Labs for the filtering system.
